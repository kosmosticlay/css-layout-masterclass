## ✅ Study Review

- 공부한 속성 리스트

  - `text-transform : uppercase` <br/>
    : 문자열을 대문자로 변환

  - `grid-template-columns: repeat(4, 1fr) 1.5fr repeat(6, 1fr)` <br/>
    : 11개로 나누어진 grid container가 존재할 때, 중간에 존재하는 grid item의 크기만 변화
