## ✅ Study Review

- 학습 내용

  - `:not(selector)` : 부정 CSS 가상 클래스; 즉 selector을 제외한 모든 요소를 선택
    - selector자리에 `*` 나 다른 부정 선택자 포함 불가 <br/>
      ```css
      div:not(.className) {
        color: black;
      }
      div:not(a) {
        color: black;
      }
      ```
  - `&nbsp;` : html 내에서 빈 칸 생성 (non-breaking space)
